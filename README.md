# WP Html Helper
The WP HTML Helper contains functions that assist in working with HTML

# Available Functions:

### The following functions are available:

## create HTML <img /> tags
```php
img_tag(
	array(
		'url' 	 => '',
		'alt' 	 => '',
		'class'  => '',
		'id' 	 => '',
		'width'  => '',
		'height' => '',
		'srcset' => ''
	)
);

```
